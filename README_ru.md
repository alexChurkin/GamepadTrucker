# Truck Remote Server V2 — геймпад + гироскоп

Управление **Euro Truck Simulator 2 / American Truck Simulator** геймпадом
**DualShock 4** или **DualSense** по **Bluetooth** (или USB). **Гироскоп — это
руль**: держи геймпад как руль и наклоняй.

---

## Как это работает

Всё идёт через **одно устройство vJoy**. Так как vJoy — обычное DirectInput-
устройство, ETS2/ATS считает его ось руля **рулём** → полный ход в обе стороны
на любой скорости, без геймпадного ассиста подруливания:

- наклон гироскопа → **руль** (ось X, сглажено, полный ход)
- R2 / L2 → **газ / тормоз** (оси Z / RZ)
- правый стик → **обзор камеры** (оси RX / RY)
- кнопки и крестовина → **кнопки vJoy** (16)

## Что нужно

- Драйвер **vJoy**. Приложение ставит и настраивает его автоматически, если его
  нет (`run.bat` → `setup_vjoy.ps1`).
- **DualShock 4** или **DualSense** по Bluetooth или USB.
- Из исходников — **Python 3.9+**. Для готового `.exe` — ничего.

## Запуск

- **Готовый:** двойной клик по `dist\TruckRemoteServerV2.exe`
  (если vJoy нет — поставь один раз через `setup_vjoy.ps1`).
- **Из исходников:** двойной клик по **`run.bat`** — создаст venv, поставит
  зависимости, при необходимости установит/настроит vJoy и запустит приложение.

## Сборка .exe

```powershell
powershell -ExecutionPolicy Bypass -File build.ps1
```

Один файл `dist\TruckRemoteServerV2.exe` (PyInstaller). `hidapi` и
`vJoyInterface.dll` из pyvjoy упакованы; на целевом ПК нужен драйвер **vJoy**.

## Настройка игры (один раз)

Назначь в игре **оси vJoy**; **кнопки настраивать не нужно** — они эмулируют
стандартные клавиши ETS2/ATS:

1. ETS2/ATS → *Настройки → Управление* → выбери устройство **vJoy**.
2. Назначь оси: **Руль → X**, **Газ → Z**, **Тормоз → RZ**,
   **Обзор камеры → RX / RY**. (Руль на стик геймпада не вешай.)
3. Кнопки уже работают через стандартные клавиши:

| Кнопка | Действие | Клавиша |
|--------|----------|---------|
| L1 / R1 | Поворотники | `,` / `.` |
| Квадрат | Стояночный тормоз | Space |
| Треугольник | Свет (цикл) | L |
| Круг | Дальний свет | K |
| Крест | Гудок | H |
| R3 | Тифон | N |
| L3 | Дворники | P |
| Options | Круиз-контроль | C |
| Share | Аварийка | F |
| Тачпад | Двигатель вкл/выкл | E |
| Крестовина | Виды камеры | 1 / 2 / 4 / 5 |
| PS | Меню | Esc |

Руль — ось vJoy, поэтому полный ход на любой скорости. Если какая-то клавиша
выше не совпадает с твоими бинды клавиатуры — поменяй её в управлении игры.

## Настройка руля (в приложении)

- **Recenter wheel** — держи геймпад в нейтрали и нажми.
- **Sensitivity** — какой поворот даёт полный ход.
- **Smoothing** — выше = плавнее, но с задержкой (гасит дрожь).
- **Dead zone** — игнор малых поворотов у центра.
- **Expo** — мягче около центра (спокойнее), полный ход остаётся достижим.
- **Invert steering** — поменять лево/право.
- **Right stick = camera look** — переключатель, со своей мёртвой зоной.
- **Reset to defaults** — вернуть подобранные стандартные настройки.

Настройки сохраняются в `settings.json` рядом с приложением.
